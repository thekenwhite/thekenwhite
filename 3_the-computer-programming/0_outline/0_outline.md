
# Outline

솔직히 고민해 봤자 어쩔 방도도 없는 인간관계에 시간을 투자하기보단, 차라리 고민하다 보면 어쩔 방도가 있는 컴퓨터에 시간을 더 투자하는 것을 권장한다.

컴퓨터가 그리 중요하냐 싶을 수도 있긴 한데, 당신이 사용하는 SNS, 스마트폰, 컴퓨터, 태블릿, 그리고 온갖 디지털 시스템 및 서버 및 인프라.

그거 전부 다 컴퓨터다.

시간을 들일 가치는 충분히 있다.

컴퓨터 프로그래밍이란 건 이제 단순히 개발자만 할 일은 아니라고 생각한다.

이런 게 위안이 될지도 모르겠지만, 어차피 개발자 지망생들도 어지간히 코딩 한 걸로는 취업도 못하고 있는 실정이다.

그러니까 그냥, 컴퓨터 프로그래밍은 취미로 두기에 딱이다.

본 직장은, 차라리 다른 직장이나 알아보는 것을 추천한다.

어차피 집에서 필요에 의해서 하는 코딩 내용과, 회사, IT 기업 등에 들어가서 하는 코딩의 내용도 상당히 다르다.

코딩으로 커리어 패스를 잡지 마라. 그것은 추천하지 않는다.
